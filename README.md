# Real Cost (MVP)

Mobile-first web app that computes the *true* lifetime cost of a purchase:
- price + tax
- financing (optional)
- maintenance over lifespan
- resale value
- opportunity cost (optional, simple model)

## Run locally

1) Install Node.js (18+ recommended)
2) In this folder:

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy (optional)

- Push to GitHub
- Import into Vercel (Next.js) and deploy
